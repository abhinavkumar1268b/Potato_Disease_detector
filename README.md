# 🥔 Potato Disease Classification App

A deep learning-based web application that detects potato leaf diseases using a Convolutional Neural Network (CNN) and Streamlit.

---

## 🚀 Project Overview

This project classifies potato leaf images into different categories:

- 🌿 Healthy
- 🍂 Early Blight
- 🍁 Late Blight
- 🚫 Not Potato Leaf (for invalid inputs)

Users can upload an image, and the model predicts whether the leaf is healthy or affected by disease.

---

## 🎯 Features

- 📸 Upload potato leaf images
- 🤖 Deep learning-based prediction (CNN)
- ⚠️ Detects invalid inputs (non-leaf images)
- 📊 Shows prediction confidence
- 🎨 Clean Streamlit UI
- 🌐 Ready for deployment (Streamlit Cloud)

---

## 🧠 Model Details

- Model Type: Convolutional Neural Network (CNN)
- Framework: TensorFlow / Keras
- Input Size: `256 x 256 x 3`
- Output Classes:
  - Early Blight
  - Late Blight
  - Healthy
  - Not Potato Leaf
