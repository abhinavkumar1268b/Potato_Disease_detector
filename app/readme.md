# Potato Disease Classification

This project is a deep learning based potato leaf disease classification system.

## Features
- Upload potato leaf image
- Predict disease class
- Detect if leaf is healthy or diseased
- Built with TensorFlow and Streamlit

## Classes
- Early Blight
- Late Blight
- Healthy

## Tech Stack
- Python
- TensorFlow
- Streamlit
- NumPy
- Pillow
